# Scripts

### term_scraper
#### Description
This script scrapes [https://www.fs.usda.gov/nwacfire/home/terminology.html]() for wildfire related terminology and generates a csv with the results. 

#### Running
To run this script you need to install the dependancies found in `scripts/term_scraper/requirements.txt`. Once that is done you can run `python3 term_scraper.py`.
